# Glossaire (terminologie Microsoft en français)

Référence pour aligner les guides sur le style Microsoft et les libellés Learn `fr-fr`.

| Terme / concept | Français recommandé | Notes |
|-----------------|---------------------|--------|
| Subscription | Abonnement | |
| Resource group | Groupe de ressources | |
| Microsoft Entra ID | Microsoft Entra ID | Nom de produit inchangé |
| Azure RBAC | Contrôle d’accès Azure (RBAC) | Garder souvent « Azure RBAC » |
| Availability zone | Zone de disponibilité | |
| Region | Région | |
| Management group | Groupe d’administration | |
| Storage account | Compte de stockage | |
| Virtual machine | Machine virtuelle | |
| Load balancer | Équilibreur de charge | |
| Application Gateway | Application Gateway | Nom de produit |
| Key Vault | Key Vault | |
| Monitor / monitoring | Supervision / supervision | Contexte observabilité |
| Pipeline (DevOps) | Pipeline | Terme courant en français IT |
| Release / deployment | Mise en production / déploiement | |
| Pull request | Demande de tirage (ou « pull request ») | Les deux existent sur Learn |

Les noms de services Azure, GitHub et Azure DevOps suivent en général les formes utilisées sur [Microsoft Learn (fr-fr)](https://learn.microsoft.com/fr-fr/).
